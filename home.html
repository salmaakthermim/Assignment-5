Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each object corresponds to a part of the document, such as elements, attributes, and text.

How do you select an element from the DOM?

Selects an element by its unique id attribute.

What is event delegaton in the context of the DOM, ans why is it useful?

Event delegation in the context of the DOM is a technique where a single event listener is added to a parent element to handle events for all of its child elements, rather than adding separate event listeners to each child element individually.
